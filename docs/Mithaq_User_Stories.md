# Mithaq — User Stories
### Tuwaiq Academy | Blockchain Contract Documentation Platform
#### Team: Araa · Rimas · Ghadi · Reem — 2026

Format: "As a [role], I want to [action] so that [benefit]."
Each story is tagged [MVP] (must-have) or [Stretch] (nice-to-have).

---

## User Stories

[MVP] As a user, I want to sign up with my email, full name, national ID, mobile number, and password so that I can create an account on the platform.

[MVP] As a user, I want to log in with my email and password so that I can access my account and contracts.

[MVP] As a user, I want to log out of my account so that my account stays secure.

[MVP] As a user, I want to view my profile so that I can see my personal and contact information.

[MVP] As a user, I want to edit my profile information so that I can keep my details up to date.

[MVP] As a user, I want to upload a profile picture so that my account feels more personal.

[MVP] As a user, I want to create a new contract from a blank form or a ready-made template so that I can start documenting an agreement.

[MVP] As a user, I want to send a contract to another party so that they can review and either accept or reject it.

[MVP] As a user, I want to accept a contract sent to me so that the agreement is officially confirmed.

[MVP] As a user, I want to reject a contract sent to me so that the other party knows I do not agree to the terms.

[MVP] As a user, I want to confirm my agreement by entering my full name as a digital signature so that the contract is officially finalized.

[MVP] As a user, I want the signed contract hash to be recorded on the blockchain so that the contract is tamper-proof and permanently verifiable.

[MVP] As a user, I want to download a PDF copy of the completed contract so that I can keep an offline record.

[MVP] As a user, I want to view all my contracts (active, pending, completed) on a dashboard so that I can track everything in one place.

[MVP] As a user, I want to filter my contracts by status or date so that I can find what I'm looking for quickly.

[MVP] As a user, I want to receive notifications for pending signatures and contract updates so that I am always up to date.

[MVP] As a user, I want to view my current subscription plan and usage so that I know how many contracts I have remaining.

[MVP] As a user, I want to get a free trial contract upon registration so that I can explore the platform before committing to a paid plan.

[MVP] As an invited party, I want to accept an invitation link so that I can join a contract and review it.

[MVP] As a user, I want every action on a contract to be logged in an audit trail so that there is a transparent record of all events.

[MVP] As any user, I want to verify a contract's authenticity via a public link so that I can confirm the contract has not been tampered with.

[Stretch] As a user, I want to switch the platform language between Arabic and English so that I can use it in my preferred language.

[Stretch] As a user, I want to toggle between dark and light mode so that I can use the platform comfortably.

[Stretch] As a user, I want to change my password from the settings page so that I can keep my account secure.

[Stretch] As a user, I want to upgrade my subscription plan so that I can create more contracts.

---

## Admin

The platform admin manages all data (users, contracts, subscriptions, plans)
directly through the Django admin panel (/admin/) using superuser credentials.
No separate admin interface is required.
